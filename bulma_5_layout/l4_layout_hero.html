hero

hero es una utilidad que se utilizará dentro de landing pages

hero est una section de clase hero, que incorpora un div de clase hero-body

class="hero"

subtitulo, dentro de hero-body dentro de hero

se pueden cambiar el tipo de hero que es el cada uno de estos elementos usando una subclase

class="hero is-primary"

Primary subtitle

class="hero is-link"

Link subtitle

class="hero is-info"

Info subtitle

class="hero is-success"

Success subtitle

class="hero is-warning"

Warning subtitle

class="hero is-danger"

Danger subtitle

por defecto podemos añadir clases adicionales para cambiar el tamaño del elemento hero

class="hero is-small is-primary"

Small subtitle

class="hero is-medium is-link"

Medium subtitle

class="hero is-large is-info"

Large subtitle

class="hero is-succes is-halfheight"

Half height subtitle

class="hero is-danger is-fullheight"

Fullheight subtitle

en caso se use una navbar, tambien se puuede indicar un fullheight sin el navbar

partes del elemento hero

el elelmento hero tiene lass siguientes subpartes por defecto

  • hero-head siempre arriba
  • hero-body siempre verticalmente centrado
  • hero-foot siempre abajo
