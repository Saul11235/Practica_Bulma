Container

container es un div que centra el contenido horizontalmente
puede ser un hijo de los siguientes elementos

  • navbar
  • heronavbar
  • section
  • footer

este container tiene dentro un subdiv class="notification is-primary"
This container is centered on desktop and larger viewports.

colocando modificadores

igual al anterior, pero el container es class="container is-widescreen"
This container is fullwidth until the $widescreen breakpoint.
igual al anterior, pero el container es class="container is-fullhd"
This container is fullwidth until the $fullhd breakpoint.
igual al anterior, pero el container es class="container is-max-desktop"
This container has a max-width of $desktop - $container-offset on widescreen and fullhd.
igual al anterior, pero el container es class="container is-max-widescreen"
This container has a max-width of $widescreen - $container-offset on fullhd.
